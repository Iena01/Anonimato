DNS Leak

DNS(Domain Name System) sono dei servizi che si occupano di effettuare la traduzione del dominio.
E possibile, in certe situazioni che in rete anonima, l'O.S contiunui ad analizzare i DNS di default del proprio ISP.
Può succedere che anche cambiando i propri DNS vengono comunque letti dal proprio ISP poiché alcuni sfruttano i proxy DNS trasparenti.

Come difendersi

Per difenderti dal DNS Leak del tuo ISP devi fare in modo che utillizi DNS pubblici o della VPN.
La soluzione migliore restano i DNS pubblici poiché garantiscono privacy, anonimato e richieste più veloci. Queste sono le piu utillizate:

  • Google DNS server primario 8.8.8.8 server secondario 8.8.4.4
  • Open DNS server primario 208..67.220.123 server secondario 208.67.220.123
  • Comodo DNS server primario 8.26.56.26 server secondario 8.20.247.20

Ricordati di fare queste due cose:

  1. Modifica i DNS del router
  2. Modifica i DNS del tuo O.S.

Fix post-cambio DNS.

Windows

$ ipconfig/flushdns---->Pulizia del dns

$ net stop dns cache---->ripulire cache in automatico

$ net start dns cache--->ripulire cache in auto

Linux

$ sudo apt-get install nscd

$ sudo etc/init.d/nscd restart

MacOs/Osx

$ sudo dscacheutil-flushcache

$ sudo killal-HUP mDNSResponder

Kill switch(protezione alla caduta di connessione)

Le Kill Switch è una funzione vitale che effetua un taglio alla connessione quallora la VPN smettesse di funzionare.

Clearnet e Deep Web

Fino ad ora abbiamo parlato di come navigare anonimi dentro la Clearnet. Nel corso degli anni si è avuta l'esigenza di navigare in una rete che non fosse scansionabile dai motori di ricerca comuni questa si chiama Deep Web.
Molti collegano il Deep Web come la parte malvagia di internet, quando in realta quella parte di internet e chiamata Darknet.
